## 数组
1. 数组的初始化
    1. 一维数组
        1. 动态初始化,必须给一个长度,只能在右边
            - int[] arr = new int[5];
        2. 静态初始化,必须给定内容
            - String[] arr = new String[]{"1","2"};
            - String[] arr = {"1","2"};
        3. 注意点:不能既给长度又给内容
            - int[] arr = new int[5]{1,2,3,4,5};这是错的
    2. 二维数组
        1. 动态初始化
            - int[][] arr = new int[2][3];
            - int[] arr[] 也行
            - int[][] arr2 = new int[2][];可以只声明行数
        2. 静态初始化
            - String[][] arr3 = {{"1","2","3"},{"4","5","6"}};
